# Site Centre De Calcul

## Requirements

PHP >= v5.6.16

MYSQL >= v4.5.2

composer


## Installation

### Step 1 : setup environment variables

#### app variables

    APP_KEY=
    APP_URL=

#### database variables

    DB_CONNECTION=(default mysql)
    DB_HOST=
    DB_PORT=
    DB_DATABASE=
    DB_USERNAME=
    DB_PASSWORD=

#### Owner variable

    OWNER_EMAIL=naimo@gmail.com

### Step 2 : install application dependency

     run command :

     $  composer install

### Step 3 : setup database tables

    run command :

     $  php artisan migrate
